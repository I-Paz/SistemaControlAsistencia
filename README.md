# Sistema de Control de Asistencias

Proyecto escolar para la materia **Fundamentos de Ingeniería de Software**.
Software de escritorio desarrollado en **Java (Swing)** y **SQL Server**.

## Características
* **Registro Rápido:** Interfaz simplificada para marcar Entrada/Salida solo con ID.
* **Gestión de Empleados:** Alta y baja lógica de personal.
* **Seguridad:** Panel administrativo protegido por contraseña.
* **Reportes:** Visualización de historial de asistencias con filtros automáticos.

## Tecnologías Utilizadas
* **Lenguaje:** Java (JDK 1.8)
* **IDE:** Apache NetBeans
* **Base de Datos:** SQL Server Express 2019
* **Driver JDBC:** mssql-jdbc-13.2.1.jre8

## Instalación y Uso
1. Ejecutar el script `BaseDeDatos.sql` en SQL Server Management Studio.
2. Crear el usuario `proyectoUser` con contraseña `123456` en SQL Server.
3. Abrir el proyecto en NetBeans.
4. Agregar el driver `.jar` a las librerías.
5. Ejecutar el archivo `VentanaAsistencia.java`.
